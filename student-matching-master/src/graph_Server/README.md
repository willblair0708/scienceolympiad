# Graph Server API

# GET Methods:
```
graph_server:5002/students `return all students`
graph_server:5002/students/<student_id> `return the given student`
graph_server:5002/students/mentors `return all mentors`
graph_server:5002/students/mentees `return all mentees`
graph_server:5002/groups/<student_id> `return all students that belong to the same group of the given student
```
